  PRACTICAS ITM es un portal web completo para gestionar estudiantes en
  prácticas universitarias, con integración de 32 programas académicos del
  Instituto Tecnológico Metropolitano.

┌─────────────────────────────────────────────────────────────────────────────┐
│ 🚀 CÓMO USAR EL SISTEMA                                                     │
└─────────────────────────────────────────────────────────────────────────────┘

  🎯 OPCIÓN 1: Inicio Rápido (Recomendado)
  ────────────────────────────────────────
  $ python main.py
  → Servidor inicia en http://localhost:5000/
  → Sistema ya tiene 32 programas cargados
  → Listo para usar

  🎯 OPCIÓN 2: Recargar Todo Desde Cero
  ──────────────────────────────────────
  $ python -c "from src.database.connection import init_db; init_db()"
  $ python load_programas.py
  $ python main.py
  → BD reinicializada
  → Programas recargados
  → Servidor iniciado

  🎯 OPCIÓN 3: Ver Demostración
  ─────────────────────────────
  $ python demo_programas.py
  → Muestra ejemplos de API
  → Filtra por criterios
  → Exporta a JSON

  🎯 OPCIÓN 4: Ejecutar Tests
  ────────────────────────────
  $ pytest tests/
  → Ejecuta todos los tests
  → Verifica integridad
  → Muestra resultados

┌─────────────────────────────────────────────────────────────────────────────┐
│ 📡 EJEMPLOS DE API (Curl)                                                   │
└─────────────────────────────────────────────────────────────────────────────┘

  🔍 Obtener todos los programas
  curl http://localhost:5000/api/programas

  🔍 Obtener un programa específico
  curl http://localhost:5000/api/programas/1

  🔍 Filtrar por facultad
  curl 'http://localhost:5000/api/programas?facultad_id=1'

  🔍 Filtrar por nivel
  curl 'http://localhost:5000/api/programas?nivel=Ingeniería'

  🔍 Solo acreditados
  curl http://localhost:5000/api/programas/acreditados

  🔍 Solo virtuales
  curl http://localhost:5000/api/programas/virtuales

  🔍 Ver estadísticas
  curl http://localhost:5000/api/programas/estadisticas

  🔍 Crear nuevo programa
  curl -X POST http://localhost:5000/api/programas \
       -H "Content-Type: application/json" \
       -d '{"nombre":"Mi Programa","nivel":"Profesional","facultad_id":1}'

  🔍 Listar estudiantes
  curl http://localhost:5000/api/estudiantes

  🔍 Ver salud del sistema
  curl http://localhost:5000/api/health
