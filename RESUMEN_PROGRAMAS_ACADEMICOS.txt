## 📞 Soporte Rápido

### ¿Cómo agregar un nuevo programa?

**Opción 1: Por CSV**
Añadir fila a `PROGRAMAS_ITM.csv` y ejecutar:
```bash
python load_programas.py
```

**Opción 2: Por API**
```bash
curl -X POST http://localhost:5000/api/programas \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Mi Programa",
    "nivel": "Profesional",
    "facultad_id": 1,
    "duracion": "8 semestres",
    "acreditada": true
  }'
```

**Opción 3: Por Base de Datos**
```python
from src.services.programa_service import ProgramaService
programa = ProgramaService.crear_programa(
    db, 
    nombre="Mi Programa",
    nivel="Profesional",
    facultad_id=1
)
```

### ¿Cómo ver programas de una facultad?

```python
# En Python
programas = ProgramaService.obtener_todos_programas(db, facultad_id=1)

# En API
curl 'http://localhost:5000/api/programas?facultad_id=1'
```

### ¿Cómo actualizar un programa?

```python
# En Python
ProgramaService.actualizar_programa(
    db, 
    programa_id=1, 
    acreditada=True
)

# En API
curl -X PUT http://localhost:5000/api/programas/1 \
  -H "Content-Type: application/json" \
  -d '{"acreditada": true}'
```
