================================================================================
                           PRIMEROS PASOS RECOMENDADOS
================================================================================

1. Leer QUICKSTART.md (5 minutos)
2. Instalar dependencias o usar Docker (2 minutos)
3. Inicializar base de datos (1 minuto)
4. Ejecutar servidor (1 minuto)
5. Hacer primer request a http://localhost:5000/api/health
6. Crear una facultad (POST /api/facultades/)
7. Crear una carrera (POST /api/carreras/)
8. Registrar un estudiante (POST /api/estudiantes/)
9. Cambiar estado a "Contratado" (PUT /api/estudiantes/{id}/estado)
10. Ver estadísticas (GET /api/estudiantes/estadisticas/facultad/{id})

Tiempo total estimado: 20 minutos

================================================================================
                          ESTADOS DE PRÁCTICA
================================================================================

🟢 DISPONIBLE
   → Estudiante disponible para ser contratado
   → Estado inicial automático

🟡 CONTRATADO
   → Estudiante en empresa realizando práctica
   → Empresa supervisa el trabajo

🔴 POR FINALIZAR
   → Práctica en recta final
   → Preparación de cierre

⚫ FINALIZÓ
   → Práctica completada
   → Archivable para reportes

Transición:
   DISPONIBLE → CONTRATADO → POR FINALIZAR → FINALIZÓ
   (No reversible, progresión lineal)
