  ✅ Validación de Campos Requeridos
     • El formulario no permite guardar sin llenar campos (*)
     • Mensaje de error si falta algún campo

  ✅ Validación de Formato
     • Email: Validación de formato correcto
     • Teléfono: Solo números
     • Documento: Solo números

  ✅ Validación de Unicidad
     • Documento: No pueden duplicarse
     • Email: No pueden duplicarse

  ✅ Validación de Dependencias
     • Carrera: Filtrada por Facultad seleccionada
     • Discapacidad personalizada: Solo si selecciona "Otra"


FUNCIONES JAVASCRIPT RELACIONADAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ openAddEstudianteModal()
     → Abre el modal de agregar estudiante
     → Carga datos de facultades y carreras
     → Inicializa el formulario vacío

  ✅ updateCarrerasFilter()
     → Se ejecuta al cambiar Facultad
     → Filtra carreras por facultad seleccionada
     → Actualiza opciones del select de Carrera

  ✅ toggleDiscapacidadPersonalizada()
     → Se ejecuta al cambiar "Presenta discapacidad"
     → Muestra/oculta campo de texto personalizado
     → Comportamiento: display flex/none

  ✅ submitForm(type, id)
     → Valida el formulario
     → Envía datos al API
     → Muestra notificación de éxito/error
     → Recarga la lista
