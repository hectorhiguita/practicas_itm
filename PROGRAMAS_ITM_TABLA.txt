🔗 INTEGRACIÓN CON EL SISTEMA
════════════════════════════════════════════════════════════════════════════════

Los programas se encuentran integrados en:

  • Sistema de Gestión de Prácticas ITM (Dashboard web)
  • Base de datos PostgreSQL (tabla carreras)
  • Formularios de inscripción de estudiantes
  • Reportes y estadísticas

Para agregar nuevos programas:
  1. Acceder al Dashboard
  2. Sección "Carreras"
  3. Seleccionar facultad correspondiente
  4. Click "Agregar Nueva Carrera"
